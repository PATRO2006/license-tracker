Client logos for the floating login bubbles.

Drop a square PNG (or JPG) here named after the client's username/slug, e.g.:
  fittr.png
  biopeak.png
  gsp-crop.png
  machaxi.png
  tech-japan.png
  quantana.png
  sunfan.png
  invarsys.png

The login page loads /logos/<slug>.png automatically. If a file is missing,
that bubble shows the client's initials instead (e.g. "FT" for Fittr).

Tip: square images (e.g. 128x128 or 256x256) look best — they're cropped to a
rounded app-icon tile.
